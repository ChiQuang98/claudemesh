# ClaudeMesh — Technical Project Summary

## Goal
Build a **Claude-native, agent-oriented framework** that allows developers to **define, register, and orchestrate custom skills and domain-specific agents** to solve complex tasks programmatically.

ClaudeMesh focuses purely on **technology, architecture, and coding**, not UI or product features.

---

## Core Concepts

### 1. Skill
- Atomic, reusable capability
- Stateless and deterministic
- Can be prompt-based or tool-backed
- Has explicit input/output schemas
