# @claudemesh/database-optimization

General SQL database optimization agents and skills for PostgreSQL, MySQL, and SQL Server.

## Agents

- **query-optimizer** - SQL query optimization and performance tuning
- **schema-designer** - Database schema design and normalization
- **index-expert** - Index strategy and optimization

## Skills

- **query-patterns** - Common SQL optimization patterns
- **indexing-strategies** - When and how to create indexes
- **migration-patterns** - Safe database migration strategies

## Installation

```bash
claudemesh add database-optimization
```

## Usage

- "Optimize this SQL query" → Uses query-optimizer
- "Design a database schema for..." → Uses schema-designer
- "What indexes should I create?" → Uses index-expert
