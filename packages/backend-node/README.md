# @claudemesh/backend-node

Backend development agents and skills for Node.js and TypeScript.

## Agents

- **api-architect** - REST/GraphQL API design expert
- **auth-specialist** - Authentication and authorization expert
- **database-integration** - ORM and database patterns expert
- **microservices-expert** - Microservices architecture expert

## Skills

- **express-patterns** - Express.js best practices and patterns
- **nestjs-patterns** - NestJS architecture and patterns
- **api-security** - API security best practices
- **error-handling** - Error handling patterns

## Installation

```bash
claudemesh add backend-node
```

## Usage

Once installed, Claude Code will automatically use these agents when appropriate:

- "Design an API for user management" → Uses api-architect
- "How should I implement JWT auth?" → Uses auth-specialist
- "Show me Express middleware patterns" → Uses express-patterns skill
