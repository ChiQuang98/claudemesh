# @claudemesh/database-athena

AWS Athena optimization agents and skills for query performance and cost optimization.

## Agents

- **athena-expert** - Athena-specific query optimization and best practices

## Skills

- **athena-partitioning** - Partition design and optimization strategies
- **athena-file-formats** - File format selection (Parquet, ORC, Avro)
- **athena-cost-optimization** - Query cost reduction techniques

## Installation

```bash
claudemesh add database-athena
```

## Usage

- "Optimize this Athena query" → Uses athena-expert
- "How should I partition my data?" → Uses athena-partitioning skill
- "Reduce Athena query costs" → Uses athena-cost-optimization skill
